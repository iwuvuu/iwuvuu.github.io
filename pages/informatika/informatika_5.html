Software

Software je kolekce instrukcí a programů, které umožňují uživateli komunikovat s počítačem. Dělí se podle využití a úrovně na jaké pracuje s hardwarem počítače.

Systémový software

Systémový software umožňuje uživateli pracovat s počítačem, ovládat ho, nebo třeba spouštět aplikace. Je přímo závislý na hardwaru a počítači.

1. Operační systém

Operační systém je základní nástroj, který umožňuje běh programů a ovlivňuje komunikaci mezi uživatelem a počítačem. Poskytuje funkce pro ovládání hardwaru, poskytuje jedodušší vývoj uživatelských programům, spravuje prostředky počítačového systému (CPU, RAM, přístup k uložišti…) a poskytuje uživatelské rozhraní (UI).

Operační systém je vhodné využívat jen tam, kde je potřeba (v kávovaru bude jen na obtíž). Hojně se využívá v:

Desktop - stolní počítače, laptopy - slouží uživateli k práci s počítačem a měl by nabízet vhodné prostředí k plnění uživatelských potřeb. Příklady: Microsoft Windows, MacOS, Linuxové distribuce, BSD distribuce.

Mobilní zařízení - vyžaduje schopnost plnit požadavky uživatele za chodu - velká efektivnost, jednoduchá obsluha. Příklady: Android, iOS, Linux na mobil (zatím docela mid).

Severy - cílem je nabídnout vysokou výpočetní techniku, snadnou údržbu a zabezpečení. Příklady: Linux, Windows NT

2. Firmware

Firmware je obecně termín užívaný k označení nízkoúrovňového softwaru, který je spjat s určitým hardwarem - Wi-Fi čipy, GPS čipy, semafory, pevné disky… Využívá se právě tam, kde velký operáční systém je nevhodný.

3. Pomocné nástroje (Utility software)

Jsou počítačové programy, které slouží k usnandění užívání počítače a zlepšení quality of life. Příkladem mohou být: správa uživatelů, clipboard managery, cryptografické programy, data recovery, kompresní programy…

Aplikační software

Aplikační software je určený pro uživatelské úkoly, které nesouvisí se správou systému. Poskytují tak práci, která není závislá na počítači.

Mezi kategorie aplikačního softwaru patří například:

Kancelářské aplikace - textové editory (MS Word, LibreOffce Writer), tabulkové procesory (MS Excel, LibreOffice Calc), prezentační programy (MS Powerpoint, LibreOffice Impress)

Multimediální software - slouží k přehrávání a úpravě médií. VLC, MPV… k úpravám lze využít Adobe Premiere, Kden Live, Audacity…

Grafický software - využití k vytváření grafických prací - 3D modelování (zahrnul bych sem i CAD software př. AutoCAD, FreeCAD), Rastrová/Vektorová grafika - GIMP, Inkscape, Photoshop, Krita

Internetový software - prohlížeče (Google Chrome, Firefox, Librewolf, Ladybug, MS Edge), e-mail klienty (Thunderbird)…

A další.

Software občas mívá zabudovanou lokalizovanou verzi, tedy překlad do jiného jazyka (bývá v jednom softwarovém balíčku. Popřípadě naprosto jinou lokalizační verzi například pro jinou orientaci čtení, jiné barvy, ikony vzhled určené pro jiné země.

Další vlastnosti, které může software mít je zlepšenou přístupnost pro různé typy lidí se zdravotním postižením. Příkladně čtečky obrazovky pro nevidomé spojené s alternativním textem obrázků. Zvýšený kontrast barev, zvětšení textu nebo titulky/přepis zvuku pro neslyšící.

Licenční politika

Softwarová licence je pravní nástroj, který umožňuje používat, upravovat nebo redistribuovat software za stanovených podmínek.

Druhy licencí

Základní dělení licencí může být podle způsobu distribuce daného softwaru.

1. Proprietární software (Copyright)

Proprietární nebo-li uzavřený software omezuje uživatelské pravomoce nad správou daného softwaru. Zamezuje uživateli možnost volně šířit a upravovat software. Poskytuje tak nejvyšší možnou ochranu autorských práv pro vývojáře.

Mezi proprietární software patří například: Microsoft Windows, MacOS, Microsoft Office, Adobe Photoshop, WinRAR…

2. Free and Open Source software FOSS

FOSS software má ve směs otevřený zdrojový kód a lze jej upravovat a redistribuovat. Avšak některé FOSS licence udávají různé podmínky a uživatelská práva.

GNU General Public License (Copyleft)

GPL (a jeji podverze -> GPL-V3; GPL-V2; LGPL…) je copyleftová licence, což znamená, že jakýkoliv software s touto licencí musí být redistribuován pod stejnou licencí a stejnými podmínkami, stejně tak všechny knihovny a programy využívány tímto programem. Umožňuje uživateli využívat a modifikovat software jakkoliv je mu libo (samozřejmě krom licence).

GPL využívá například: Linux, Git, GIMP… ***

MIT license

Narozdíl od GPL licence MIT není striktně copyleftová a neomezuje jaké licence musí být programem využity, to znamená, že program pod MIT licencí může být využit později v jiném proprietárním programu. Podobně jako GPL umožňuje uživateli modifikovat a využívat software bez omezení.

MIT využívá velké množství knihoven a programů: x-org, node.js, react…

Mezi další FOSS licence patří například Apache license (Android…), BSD license…

3. Podle způsobu distribuce

Freeware - užívání je naprosto zdarma, ale zamezuje úpravu a redistribuci

Shareware - lze volně distribuovat a zdarma vyzkoušet, pro další používání je nutné zaplatit

Commercial software - software určený pro zisk, který je nutný si koupit

Způsob distribuce nemusí ovlivnit využitou licenci, například GPL software může být commercial software a Proprietární software může být Freeware. Obvykle je však proprietární software distribuován komerčně a FOSS software distribuován bez omezení.

EULA (End User Agreement License)

EULA je koncová uživatelská licenční smlouva, která určuje podmínky (co smí nebo nesmí dělat) pro uživatele, které k využívání softwaru musí přijmout.

Software as a service (SaaS) je model, kdy se software neinstaluje na počítač, ale je dostupný přes cloud. Běží tak na serverech poskytovatele a my k němu pouze dostáváme přístup. Příkladem mohou být třeba Google Docs nebo Dropbox.

🧩 Rozpoznání a zapojení součástek na základní desce (motherboard)

✅ Základní orientace

Základní deska je hlavní deska počítače – propojuje veškerý hardware. Obsahuje socket pro CPU, sloty pro RAM, PCIe pro grafiku, konektory pro disky a porty.

🔌 1. Socket procesoru (CPU Socket)

  • Funkce: Umožňuje osazení procesoru (CPU)
  • Typické označení: AM4, AM5 (AMD), LGA1700, LGA1200 (Intel)
  • Jak poznat: Velké čtvercové místo s paticí a zajišťovací pákou
  • Na co si dát pozor:
    • Musí odpovídat typu CPU (nesprávný socket = nepasuje)
    • Při osazování neohýbat piny

🧠 2. Sloty pro RAM (DIMM)

  • Funkce: Umožňují osazení operační paměti (RAM)
  • Jak poznat: Dlouhé úzké sloty (většinou 2 nebo 4), často černé a šedé
  • Značení: DDR4 / DDR5
  • Na co si dát pozor:
    • Zasunout správným směrem (zářez musí sedět)
    • Pro dual channel RAM osazovat párové sloty (např. A2+B2)

🎮 3. PCI Express sloty (PCIe)

  • Funkce: Pro připojení rozšiřujících karet (grafická karta, Wi-Fi karta)
  • Typy: x1, x4, x16 (nejčastěji grafika = x16)
  • Jak poznat: Dlouhý slot (x16) většinou nejblíže CPU
  • Na co si dát pozor:
    • Vložit GPU do hlavního (nejrychlejšího) slotu x16
    • Zajistit kartičku šroubkem ke skříni

💾 4. Konektory pro disky (SATA a M.2)

SATA

  • Funkce: Připojení klasických SSD/HDD
  • Jak poznat: L-tvarovaný plastový konektor, obvykle po straně desky
  • Pozor: Nutno zapojit SATA napájení ze zdroje + datový kabel do desky

M.2 slot

  • Funkce: Pro připojení rychlých NVMe SSD
  • Jak poznat: Malý vodorovný konektor s šroubkem
  • Na co si dát pozor:
    • Rozlišit NVMe vs SATA M.2 disk (ne všechny M.2 sloty podporují obojí)
    • Disk musí být přišroubovaný – jinak může vypadnout

⚡ 5. Napájecí konektory

24pin ATX konektor

  • Hlavní napájení základní desky
  • Vede ze zdroje → zasunout silou až na doraz

8pin CPU (EPS)

  • Napájení procesoru
  • Umístěn u horního okraje desky poblíž CPU socketu

PCIe napájení (pro GPU)

  • Není na desce – vede ze zdroje přímo do grafické karty

🌬️ 6. Fan a Chladič konektory

  • CPU_FAN: Pro chladič procesoru – nutný pro spuštění systému
  • SYS_FAN / CHA_FAN: Pro systémové větráky
  • Jak poznat: 3–4 pinové konektory označené přímo na desce
  • Pozor: CPU_FAN musí být zapojen – jinak deska odmítne nabootovat

🧱 7. Interní I/O konektory (přední panel)

  • PWR_SW, RESET_SW, HDD_LED, PWR_LED
  • Slouží k připojení tlačítek a LEDek na předním panelu skříně
  • Pozor:
    • Zapojení bývá nepřehledné → používat manuál desky
    • Špatné zapojení = nefunkční tlačítka

📺 8. Zadní I/O panel

  • Porty: USB, LAN, HDMI, DisplayPort, zvuk
  • Většinou připájeny přímo na desku
  • Pozor:
    • Výstupy obrazu fungují jen pokud máš iGPU v CPU
    • Přístupné zezadu přes výřez ve skříni

✅ Tipy a časté chyby

⚠️ Chyba ❗ Problém ✅ Řešení
RAM špatně zasunutá Počítač nenaběhne Zkontroluj, zda je správně zacvaknutá
CPU piny ohnuté Deska nefunguje Montovat opatrně, netlačit násilím
GPU bez napájení Černá obrazovka Zkontroluj PCIe konektory ze zdroje
Chybí CPU_FAN Deska se nespustí Vždy zapojit chladič do správného konektoru
Špatné zapojení panelu Nereaguje tlačítko Použít schéma v manuálu desky
