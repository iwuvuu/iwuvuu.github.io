Jedná se o nástroj k vytváření algoritmů a programů, díky nimž říkáme počítači, jaké operace má provádět. Slouží k vytváření sofrtwaru, aplikací, webů nebo řízení hardwaru. Počítač sám našemu jazyku nerozumí, pracuje pouze v nulách a jedničkách, proto programovací jazyky slouží jako překladač mezi jazykem počítače a lidským jazykem..

Vznik

Hlavní důvod vzniku je zjednodušení komunikace lidí a počítače. Podpora složitějších projektů a zlepšení vývoje softwaru.

Dělení

Podle úrovně abstrakce

Nízkoúrovňové jazyky

  • blízký strojovému kódu -> instrukce přímo určené pro procesor zapsané pomocí posloupností číselných kódů.
  • např binární zápis
  • přímo vyžadují, aby programátor kontroloval prvky ukládání a zpracování dat
  • jsou rychlejší a efektivnější, jelikož nepotřebují mezikrok překladu a využívají maximální výkon hardwaru
  • obtížná čitelnost a psaní, což vede k větší náchylnosti k chybám
  • velmi nízká kompatibilita s jiným druhem hardwaru
  • využití:
    • operační systémy
    • firmware
    • ovladače
  • příklady:
    • strojový jazyk (binární kód)
    • Assembler

Vysoceúrovňové jazyky

  • pro člověka srozumitelnější -> algoritmy jsou zapsány v jazyce blízký lidským jazykům
    • výrazy např z angličtiny
  • umožňují tak lepší čitelnost a srozumitelnost
  • k využití programátoři nepotřebují velké hardwarové znalosti a nejsou vázané ke specifickému hardwaru
  • nejsou tak výkoné a rychlé, ale programátor je v jejich psaní daleko více efektivní
  • využití:
    • vývoj aplikací
    • weby
    • datová analýza
  • příklady:
    • C
    • Python
    • Rust
    • JavaScript

Podle programovacích paradigmat (Princip jak funguje)

  • Procedurální jazyky - kód je rozdělen do funkcí (C, Pascal)
  • Objektově orientované jazyky - pracují s objekty a třídami (Java, C++, C#)
  • Funkcionální jazyky - založené na matematických funkci (Haskell, Lisp)
  • Skriptovací jazyky - pro automatizaci a práci s webem (JavaScript, PHP)

Mikrokontrolery využívají různé programovací jazyky k jejich operaci. Například Arduino využívá standardně C/C++ a Micro:bit využívá programování přes vlastní blokový programnovací jazyk nebo pyhthon/javascript. Takto můžeme programovat přímo na mikrokontrolerech bez nutnosti znát hodně nízkoúrovňové jazyky.


Důležitou roli v programovacích jazycích hraje syntax (sada pravidel a struktur v daném jazyce). Fungují podobně jako gramatika v běžném jazyce a počítač tak pochopí, co jaký vstup znamená. Příkladně JavaScript by měl končit ‘;’, aby bylo jasné, že zde končí příkaz. String se nejčastěji zapisuje do uvozovek - počítáč tak ví, že se opravdu jedná o zadaný text a ne o proměnnou. Jisté jazyky jsou striktnější (například C++) a nedovolí opomíjení syntaxe/vyžadují přesný syntax a deklaraci všech typů atd. a jiné (třeba python nebo Javascript) jsou volnější a nemají takovou nutnost psát složitý syntax.

Programovací jazyky

Jazyk symbolických adres (Assembler)

Jedná se o nízkoúrovňový programovací jazyk, který odpovídá strojovému kódu procesoru. Programátor místo číselných instrukcí používá symbolické názvy (např MOV, ADD, JMP…), které odpovídají konkrétní operaci procesoru. Assembler umožňuje přímou kontrolu nad hardwarem (firmware, ovladače, vestavěné systémy…).

Basic (Beginner’s All Purpose Symbolic Instruction Code)

Cílem k vytvoření programovacího jazyku Basic bylo naučit začátečníky programovat. Z toho důvodu jsou instrukce zaspány v běžné angličtině a příkazy jednoduše srozumitelné. Čehož se využívá i v dnešních programovacích jazycích.

Pascal

Další programovací jazyk určený k výuce programování. Byl vytvořen tak, aby byl snadný a čitelný, a aby ho bylo snadné implementovat na většinu tehdejších počítačů. Jedná se o silně typový jazyk, který je kompilovaný do trojového kódu. Má čitelný syntax a podporuje různé datové typy a struktury.

C

Cílem vzniku Cčka bylo vyvinout jazyk vhodný pro tvorbu operačních systémů - původně určen pro UNIX. Jedná se o procedurální nízkoúrovňový jazyk (s prvky vyšší abstrakce -> středně úrovňový jazyk). Poskytuje přímý přístup k paměti a kód je velmi rychlý a efektivní.

  • Využití - hlavně k tvorbě operačních systémů, ovladačů, vestavěných systémů a databází.

  • Výhody:

    • vysoký výkon
    • kompatibilní s různými platformami
    • flexibilita a velká kontrola nad hardwarem

  • Nevýhody:

    • náročnější psaní a čtení kódu, než moderní jazyky
    • menší ochrana před chybami - nutnost správně ošetřit paměť

C++

Jedná se o rozšířenou verzi klasického C, které doplňuje o objektové orientované prvky. Jedná se o vysokoúrovňový jazyk, který je zárověň rychlý. Hlavní jeho využití je při tvorbě her, systémového softwaru i desktopových aplikací - Photoshop, MS Office, prohlížeče.

C

Vytvořen Microsoftem jako moderní jazyk pro vývoj aplikací v rámci platformy .NET. Spojuje prvky C++ a Javy a zároveň se soustředí na jednoduchost. Doplňuje C++ o některé funkce, například garbage collection, díky čemuž se programátor nemusí starat o alokování paměti. V dnešní době se využívá na vývoj her nebo třeba mobilních aplikací.

Java

Cílem Javy bylo zajistit programovací jazyk, který bude tvořit přenositelné programy, tedy napsat program jednou a spustit ho všude. Využívá se k hlavně k vývoji mobilních aplikací.

Python

Jedná se o jednoduchý, srozumitelný a univerzální programovací jazyk. Má velmi jednoduchou a čitelnou syntaxy. Má jednoduchou typizaci (nemusí předem definovat typy proměnných) a podporuje jak procedurální, tak objektové a funkcionální programování. Jedná se o velmi populární jazyk a díky tomu dnes existují spousty knihoven a modulů.

  • využití:
    • datová analýza
    • strojové učení
    • automatizace
    • skriptování
  • výhody:
    • jednoduchý
    • velká a aktivní komunita
    • platformově nezávislý
  • nevýhody:
    • pomalejší, než kompilované jazyky (C, C++) - ale i tak velmi rychlý
    • není tak vhodný pro tvorbu náročných aplikací s vysokými nároky na výkon

JavaScript

Určen pro tvorbu webových stránek, kde umožňuje interaktivní funkce. Je to také nejpoužívanější programovací jazyk na světě. Jedná se o skriptovací jazyk běžící hlavně v prohlížeči, je dynamicky typovaný a neustále vyvíjený.

  • využití:
    • front-end vývoj -> weby - animace, interaktivita…
    • back-end vývoj -> Node.js - stavění aplikací v JS
    • Mobilní a desktopové aplikace (React Native, Electron - spotify, discord…)
  • výhody:
    • nutný k tvorbě webových stránek
    • vývoj napříč platformami
  • nevýhody:
    • volné typování -> občas nepředvídatelný
    • běží v prohlížeči - bezpečnostní riziko při špatném zacházení na straně klienta - code injection

PHP

Další skriptovací jazyk využívaný k práci na webu. Slouží k ukládání, měnění a úpravě dat na webových stránkách.

Rust

Jedná se o moderní programovací jazyk, který slouží jako alternativa k C/C++ a přináší spoustu výhod. Je kompilovaný (stejně jako C) a je zaměřený na bezpečnost a výkon. Narozdíl od základní C rodiny má vlastní systém správy paměti - neobsahuje garbage collector, ale bez rizika úniku paměti. Má striktní typovací systém, přísný překladač, ale s velkou jistotou správnosti kódu.

  • využití:
    • systémové programování
    • vývoj herních enginů
    • webové servery
    • knihovny pro jiné jazyky
    • operacní systémy
    • embedded systémy
  • výhody:
    • vysoký výkon
    • bezpečné zacházení s pamětí
  • nevýhody:
    • strmá křivka učení
    • zatím méně rozšíření (but the world will be rusty)

Pro výuku algoritmů se využívá umělých jazyků. Populární je blokový programovací jazyk pro děti Scratch. Ten je postaven na propojování barevných bloků s různou funkcí, které potom společně umožní programu fungovat. Na podobném principu funguje třeba Blockly od Google.
